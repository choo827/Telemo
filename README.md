# Telemo

Telemo는 간단한 연락처 메모장으로 연락처를 저장했을 때 카카오톡에 자동 연동되는 것을 방지하기 위해 만든 앱입니다.

## 목표
Google play에 올리는 것

## 공부한 것
- Kotlin
- CircularRevealFrameLayout
- Firebase Auth (Google)
- Firestore
- Crashlytics
- Firebase Analytics
- 스토어 등록
